Psychologist Finder
Psychologist Finder is a web application designed to help users find and connect with professional psychologists. The app provides detailed profiles for each psychologist, including their experience, specialization, rating, consultation price, and user reviews.

Features:
User Authentication: Users can register, log in, and log out. Certain actions, like adding psychologists to favorites, are only available to logged-in users.
Browse Psychologists: View a list of psychologists with key information displayed.
Favorites: Logged-in users can mark psychologists as favorites for easy access later. Favorites persist across sessions.
Filtering and Sorting: Sort psychologists alphabetically (A–Z, Z–A), by price, or by rating.
Pagination: Large lists of psychologists are paginated for better navigation.
Detailed Profiles: Each psychologist has a dedicated profile with reviews and detailed information.
Protected Routes: Certain pages (like favorites) are only accessible to authenticated users.
Asynchronous Data Loading: Data is fetched from Firebase in real time.

Technologies Used:
React with Hooks
Redux Toolkit 
Firebase Authentication and Realtime Database
React Router 
CSS Modules 
React Suspense and Lazy Loading
